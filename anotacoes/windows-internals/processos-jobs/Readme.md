## Processo

Em poucas palavras, processo é uma instância de um programa em execução. O código fica em arquivo; ao executar, vira processo, que realiza o que o programa define.

## Jobs

Job é uma unidade de trabalho ou conjunto de instruções entregue ao SO. Pode incluir programas pequenos ou processos maiores, conforme o objetivo.
