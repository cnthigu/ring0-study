# cpp-cnt-study

Repositório de estudos de C++. Tudo que uso para aprender fica aqui: livros, exercícios, snippets de referência, mini-projetos, anotações e ferramentas.

| Pasta | O que é |

| [`exercicios/`](./exercicios/) | Exercícios organizados por fonte (livros, Exercism, etc.) |
| [`snippets/`](./snippets/) | Referência rápida de código |
| [`projetos/`](./projetos/) | Mini-projetos construídos do zero durante os estudos |
| [`ferramentas/`](./ferramentas/) | Setup do ambiente: WinDbg, Visual Studio, desenvolvimento de drivers |
| [`videos/`](./videos/) | Lista de vídeos e playlists |
| [`anotacoes/`](./anotacoes/) | Notas gerais, conceitos, dúvidas resolvidas |
