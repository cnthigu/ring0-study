### Tradução PT-BR

Tradução não oficial para o **português brasileiro** feita com auxílio de IA, capítulo por capítulo.
Cada capítulo inclui **exemplos de código C++ prontos para rodar no Visual Studio** que demonstram os conceitos do livro na prática.

Ideal para quem quer estudar Windows Internals em português - o livro nunca foi traduzido oficialmente.

- Processos, threads, memória virtual, I/O, segurança - tudo em PT-BR
- Exemplos práticos com WinAPI: `CreateProcess`, `VirtualAlloc`, `OpenProcessToken`, `CreateThread` e muito mais
- Termos técnicos como kernel, handle, driver, heap mantidos em inglês (padrão da área)
