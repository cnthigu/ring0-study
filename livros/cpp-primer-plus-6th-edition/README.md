### Tradução PT-BR

Tradução não oficial para o **português brasileiro** feita com auxílio de IA, capítulo por capítulo.
Cada capítulo inclui **exemplos de código C++ prontos para rodar no Visual Studio** (ou compilar com `g++` / `clang++`) que demonstram os conceitos do livro na prática.

O livro foi escrito em 2011 e cobre **C++11**. Quando algum trecho mostra código que envelheceu, a tradução adiciona blocos curtos **"Em C++ moderno (17/20/23)"** mostrando como o mesmo conceito apareceria hoje (smart pointers, `std::optional`, `std::string_view`, ranges, regra dos 5, etc).

Ideal para quem está começando em C++ ou migrando do C, e quer estudar em português - o livro nunca foi traduzido oficialmente.

- 18 capítulos cobrindo toda a linguagem: do `cout` até templates avançados, exceções e STL
- Termos técnicos como `pointer`, `reference`, `template`, `namespace`, `RAII`, `STL` mantidos em inglês (padrão da área)
- Notas pontuais de C++ moderno (17/20/23) onde o livro original mostra apenas o estilo C++11
