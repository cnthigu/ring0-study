## Sobre esta traducao

Traducao do livro **C++ Primer Plus** para o **portugues brasileiro**, feita com auxilio de IA e revisada manualmente. O livro cobre toda a linguagem C++ a partir do C++11, indo do basico (variaveis, IO, controle de fluxo) ate templates, exceptions, classes com memoria dinamica, heranca, STL e a string class.

Cada capitulo inclui **exemplos de codigo C++ prontos para rodar no Visual Studio** ou compilar com `g++` / `clang++`.

Como o livro e de 2011 e ensina ate C++11, a traducao adiciona blocos curtos **"Em C++ moderno (17/20/23)"** quando o codigo do livro envelheceu (smart pointers, ranges, `std::optional`, regra dos 5, `[[nodiscard]]`, etc).

Criterios de traducao:

- Todo texto narrativo e explicativo traduzido para PT-BR natural e fluente
- Termos tecnicos consagrados mantidos em ingles: `pointer`, `reference`, `class`, `template`, `namespace`, `inline`, `virtual`, `friend`, `STL`, `iterator`, `lambda`, `lvalue/rvalue`, `RAII`, etc.
- Nomes de funcoes/classes da std mantidos em ingles
- Palavras-chave da linguagem nao sao traduzidas
- Primeira ocorrencia de termos importantes: "termo em portugues (termo em ingles)"
